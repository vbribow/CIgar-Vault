# Document 003 — Cedriva Company Philosophy

**Version:** 0.1  
**Status:** Working Draft  
**Owner:** Brian Bowers  
**Created:** July 23, 2026  
**Last Updated:** July 23, 2026

Cedriva is not in the software business.

Cedriva is in the collector business.

We happen to build software.

Every feature should help someone become a better collector.

Every screen should teach something.

Every interaction should increase confidence.

Every collector should feel welcome.

Every expert should feel respected.

The platform should become more valuable because of its community.

Knowledge should compound.

History should be preserved.

Stories should never be lost.

Technology should disappear behind the experience.

The collector should always remain the hero.
