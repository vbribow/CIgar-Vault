# Document 014 — The Cedriva Product Test

**Version:** 0.1  
**Status:** Working Draft  
**Owner:** Brian Bowers  
**Created:** July 23, 2026  
**Last Updated:** July 23, 2026

Before any feature is approved, ask:

Would this help someone become a collector?

Would this help an experienced collector?

Would this help preserve cigar history?

Would this help grow the premium cigar community?

Would this make the hobby less intimidating?

Would this make someone proud to recommend Cedriva?

If we cannot answer **YES** to at least one of these questions, the feature should not be built.
